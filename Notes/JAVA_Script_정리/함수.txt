====== 함수 정의 ======
Created 수요일 11 7월 2018

**함수 선언**

===== 함수 구문 선언식 : function 함수이름( ) { } =====
- 함수가 정의될때 메모리에 올린다. (호이스팅) 선언될때는 안올라감.
ex:  선언 var x;   정의 var x = 7;  값을 넣어주는거..
- 메모리에 있으므로 어디에서나 사용가능하다.

-ex) 모두 실행 됨
1. console.log(myFn(x));
	function 함수이름( ){코드내용}

2. function myFn( ){코드내용}
	myfn(x);




===== 함수 표현식 (익명함수) : var myFn = function( ) { } =====
-함수가 정의 되어도 메모리에 올라가지 않음(호이스팅x)
-실행할때 메모리에 올라가므로 사용하기 전에 앞에서 입력해줘야함.
-ex) 1번 실행 안됨 2번 실행됨 
1. console.log(myFn(x));
	var myFn = function(){코드내용}

2. var myFn = function(){코드내용}
	myfn(x);


**함수에 전달되는 값 **

함수의 매개변수에 기본자료형이 들어가면 값으로 함수에 전달된다.
ex) var x =1;   myFn(x);   이때 myFn()에는 x의 값인 1이 전달되는 것이다.

그러므로 함수가 매개변수의 값을 바꾸더라도 함수를 호출한 곳에는 반영되지 않는다.
ex) var x=1; var y=myFn(x); x=3; y는 값만 전달받은것이므로 바뀌지 않고 1이 나온다.

함수의 매개변수에 객체 또는 배열이 올때는 참조값(주소)이 전달된다.




**함수의 범위**

함수 내에서 정의된 변수는 함수의 밖에서 불러서 쓸수 없다.

그러나 함수 밖에서 정의된 변수를 함수 안으로 가져와서 쓸수는 있다.
1. 부모함수의 변수를 가져다가 쓸수 있다.
2. 부모함수가 가져올수 있는 변수를 가져다가 쓸수 있다.
3. 부모함수는 자식함수의 변수를 가져다 쓸수 없다.



**재귀함수 (자기 자신을 호출하는 함수)**

함수는 자기 자신을 참조하고 호출할수 있다. (코드를 여러번 실행하고 정지조건이 있다)

1. 함수의 이름 == 1. bar()
2. arguments.callee == 2. arguments.callee()
3. 함수를 참조하는 범위 내 변수 ==  3. foo()

__var foo = function bar() { // statements go here };__

함수 본문 내에서 다음은 모두 동일합니다.

먼저 들어온게 나중에 나가는 방식이다.
