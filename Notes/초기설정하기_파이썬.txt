====== 초기설정하기 파이썬 ======
Created 금요일 15 6월 2018


**파이선 다운로드** https://www.python.org/downloads/windows/
1.파이썬 설치할때 Add Python 3.6 to PATH 꼭 체크하기
2. Customize installation 선택
3. next누르고 Advanced Opitons에서 폴더 위치를 새로 지정해준다
location : D:\Python\Python36-32 으로 지정해준다

**장고 설치하기**
1.명령프롬프트에서 d드라이브로 이동 			> d:
2. djangogirls 폴더를 생성한다 				> mkdir djangogirls)
3. 장고걸스 폴더 안에 진입 한다 				> cd djangogirls
4. 가상환경을 생성한다.					> python -m venv myvenv
5. 가상환경을 실행한다.					> myvenv\script\actvate
6. (myvenv) 가 앞에 뜨면 성공!  장고를 설치한다.	> pip install django~=1.11.0
7. pip list 입력으로 장고가 잘 설치되엇는지 확인	> pip list 
8. 실행 후에
