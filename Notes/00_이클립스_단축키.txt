====== 00 이클립스 단축키 ======
Created 화요일 11 9월 2018

줄정렬 Ctrl + Shift + f

줄이동 Alt + 방향키

한줄 복사 Alt + Ctrl + 방향키

한줄삭제 Ctrl + d

이전으로 되돌리기 Ctrl + y, Ctrl +z

import 문을 자동 생성 Ctrl + Shift + O
import 문을 자동 생성  객체에 커서를 위치 시키고 Ctrl + Shift + M

주석달기 Ctrl + Shift + / 
주석 해제  Ctrl + Shift + \

에디터모드(다중커서) Alt + Shift + A
